#Cấu trúc thư mục MVC

Đây là một ví dụ về cách tạo một RESTful API User Resource sử dụng Express, PostgreSQL, Sequelize và mô hình MVC.


```code

- app/
  - controllers/
    - userController.js
  - models/
    - index.js
    - userModel.js
  - routes/
    - userRoutes.js
  - app.js
- config/
  - config.json
- node_modules/
- package.json
- package-lock.json
```
