# Deployment

## 💛 Bước chuẩn bị

- Chạy lệnh 

```bash
yarn build
npm run build
```
Để build thử ở local và đảm bảo là không có lỗi gì xảy ra mới commit lên git.

- Đẩy code lên nhánh main

## 💛 Deploy Database

- Tạo tài khoản MongoDB Atlas, Tạo một Cluster
- Lấy Connection String dán vào phần kết nối với Mongoose


## 💛 Deploy App
