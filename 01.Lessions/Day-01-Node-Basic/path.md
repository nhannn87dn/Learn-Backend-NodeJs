# Path Module

Chi tiết: https://nodejs.org/docs/latest/api/path.html

Liên quan đến vấn đề xử lý đường dẫn cho file

Ex: lấy tên file

```js
const path = require('node:path');
const filename = path.basename('/Users/Refsnes/demo_path.js');
console.log(filename);
```
