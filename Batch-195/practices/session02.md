# Thực hành Route ExpressJs

## Task 1

Tạo route khớp với định dạng sau:

* `/contact` với method GET
* Student (với method GET)
    * `/student/edit/1` 
    * `/student/edit/2`
* User (với method GET)
    * `/user/tomyRan1234`
    * `/user/alextomJon323`

## Task 2

Tạo một Task resource 

* GET `/task`
* GET `/task/:id`
* POST `/task`
* PUT `/task/:id`
* DELETE `/task/:id`

Và data handling tương ứng với các method trên và trả về response tương ứng với JSON data.
